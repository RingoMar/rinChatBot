# Ringo Bot the chat bot
Ringomar's Python Machine Learning chatbot.
- A chatbot that is designed by humans to look like humans.

# How to run
+ Requires python 3.6 + 

`pip install -r requirements.txt`

`python train.py` 

`python main.py`
